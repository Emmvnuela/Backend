# 🍽️ Backend - Cantine Univ

API ASP.NET Core pour la gestion des étudiants, menus et réservations.

## 🔧 Technologies
- .NET 8
- MySQL + Entity Framework Core
- JWT Authentication
- Swagger

## 🚀 Démarrer le projet

```bash
dotnet build
dotnet ef database update
dotnet run --project BackendApp.csproj
